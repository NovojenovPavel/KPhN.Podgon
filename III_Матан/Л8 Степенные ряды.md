### –¢–µ–æ—Ä–µ–º–∞ –ê–±–µ–ª—è
–ï—Å–ª–∏ —Å—Ç–µ–ø–µ–Ω–Ω–æ–π —Ä—è–¥ —Å—Ö–æ–¥–∏—Ç—Å—è –≤ —Ç–æ—á–∫–µ $x_o \neq 0$, —Ç–æ –æ–Ω –∞–±—Å–æ–ª—é—Ç–Ω–æ —Å—Ö–æ–¥–∏—Ç—Å—è –ø—Ä–∏ –≤—Å–µ—Ö –∑–Ω–∞—á–µ–Ω–∏—è—Ö $x$ —É–¥–æ–≤–ª–µ—Ç–≤–æ—Ä—è—é—â–∏—Ö –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤—É:
$$|x| < |x_o|$$

–ï—Å–ª–∏ —Å—Ç–µ–ø–µ–Ω–Ω–æ–π —Ä—è–¥ —Ä–∞—Å—Ö–æ–¥–∏—Ç—Å—è –≤ —Ç–æ—á–∫–µ $x_1 \neq 0$, —Ç–æ –æ–Ω —Ä–∞—Å—Ö–æ–¥–∏—Ç—Å—è –ø—Ä–∏ –≤—Å–µ—Ö –∑–Ω–∞—á–µ–Ω–∏—è—Ö $x$ —É–¥–æ–≤–ª–µ—Ç–≤–æ—Ä—è—é—â–∏—Ö –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤—É:
$$|x| > |x_1|$$

__–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ:__
_–ü–µ—Ä–≤–∞—è —á–∞—Å—Ç—å_
$$\sum_{n=0}^\infty a_nx^n = a_o + a_1x + a_2x^2 + \dots \ (1)$$
–¢–∞–∫ –∫–∞–∫ —Å—Ç–µ–ø–µ–Ω–Ω–æ–π —Ä—è–¥ —Å—Ö–æ–¥–∏—Ç—Å—è –≤ —Ç–æ—á–∫–µ $x_o \neq 0$, —Ç–æ –≤—ã–ø–æ–ª–Ω—è—é—Ç—Å—è –Ω–µ–æ–±—Ö–æ–¥–∏–º—ã–µ —É—Å–ª–æ–≤–∏—è —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏:
$$\lim_{n \rightarrow \infty} a_nx_o^n = 0 \ (2)$$
–í–≤–µ–¥–µ–º —á–∏—Å–ª–æ $y_n$, —Ç–∞–∫–æ–µ —á—Ç–æ:
$$y_o = a_nx_o^n; \ |y_n| = |a_nx_o^n| \leq M$$

$$|a_o| + |a_1x_o|\left(\frac{|x|}{|x_o|}\right) + |a_2x_o^2|\left(\frac{|x|}{|x_o|}\right)^2 + \dots + |a_nx_o^n|\left(\frac{|x|}{|x_o|}\right)^n \ (3)$$

$$M + M\left(\frac{|x|}{|x_o|}\right) + M\left(\frac{|x|}{|x_o|}\right)^2 + \dots + M\left(\frac{|x|}{|x_o|}\right)^n \ (4)$$

–†—è–¥ 4 —Å—Ö–æ–¥–∏—Ç—Å—è –ø—Ä–∏ –∑–Ω–∞—á–µ–Ω–∏—è—Ö $x: \frac{|x|}{|x^o|} = q<1$, —Ç–∞–∫ –∫–∞–∫ —ç—Ç–æ –≥–µ–æ–º–µ—Ç—Ä–∏—á–µ—Å–∫–∞—è –ø—Ä–æ–≥—Ä–µ—Å—Å–∏—è.

–ü–æ –ø—Ä–∏–∑–Ω–∞–∫—É —Å—Ä–∞–≤–Ω–µ–Ω–∏—è —Ä—è–¥ —Å—Ö–æ–¥–∏—Ç—Å—è –ø–æ –∞–±—Å–æ–ª—é—Ç–Ω—ã–º –≤–µ–ª–∏—á–∏–Ω–∞–º.

_–í—Ç–æ—Ä–∞—è —á–∞—Å—Ç—å_

–î–æ–∫–∞–∂–µ–º –æ—Ç –ø—Ä–æ—Ç–∏–≤–Ω–æ–≥–æ. –ü—Ä–µ–¥–ø–æ–ª–æ–∂–∏–º, —á—Ç–æ –≤ —Ç–æ—á–∫–µ $x: |x| > |x_1|$, —Ä—è–¥ —Å—Ö–æ–¥–∏—Ç—Å—è. –¢–æ–≥–¥–∞ –ø–æ –ø–µ—Ä–≤–æ–π —á–∞—Å—Ç–∏ –æ–Ω –¥–æ–ª–∂–µ–Ω —Å—Ö–æ–¥–∏—Ç—å—Å—è –≤ —Ç–æ—á–∫–µ $x_1$. –°–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ —Ä—è–¥ —Ä–∞—Å—Ö–æ–¥–∏—Ç—Å—è, –¥–ª—è –≤—Å–µ—Ö $x: x > x_1$.

> –ò–∑ —Ç–µ–æ—Ä–µ–º—ã –ê–±–µ–ª—è —Å–ª–µ–¥—É–µ—Ç, —á—Ç–æ —Ä—è–¥ —Å—Ö–æ–¥–∏—Ç—Å—è –≤ –Ω–µ–∫–æ—Ç–æ—Ä–æ–º –∏–Ω—Ç–µ—Ä–≤–∞–ª–µ. –≠—Ç–æ—Ç –∏–Ω—Ç–µ—Ä–≤–∞–ª –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –∏–Ω—Ç–µ—Ä–≤–∞–ª–æ–º —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏. –°—Ö–æ–¥–∏–º–æ—Å—Ç—å —Å—Ç–µ–ø–µ–Ω–Ω–æ–≥–æ —Ä—è–¥–∞ –Ω–∞ –∫–æ–Ω—Ü–∞—Ö —ç—Ç–æ–≥–æ –æ—Ç—Ä–µ–∑–∫–∞ –∏—Å—Å–ª–µ–¥—É–µ—Ç—Å—è –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ.

### –í—ã—á–∏—Å–ª–µ–Ω–∏—è —Ä–∞–¥–∏—É—Å–∞ —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏.
1) $$u_n(x) = a_nx^n$$
2) $$|u_n(x)| = |a_n||x^n|$$
3) –ü—Ä–∏–º–µ–Ω–µ–Ω–∏–µ –ø—Ä–∏–∑–Ω–∞–∫–∞ —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏ –î–∞–ª–∞–º–±–µ—Ä–∞ –∏–ª–∏ —Ä–∞–¥–∏–∫–∞–ª—å–Ω—ã–π –ø—Ä–∏–∑–Ω–∞–∫ –ö–æ—à–∏ –∫ —Ä—è–¥—É $$|u_n(x)|$$
4) –ò–∑ —É—Å–ª–æ–≤–∏–π —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏ –Ω–∞—Ö–æ–¥–∏–º $R$

–ü—Ä–∏–º–µ—Ä:
$$
\lim_{n \rightarrow\infty}\sqrt[n]{|u_n(x)|} = \lim_{n \rightarrow\infty}\sqrt[n]{|a_n||x|} = \lim_{n \rightarrow\infty}|x|\sqrt[n]{|a_n|} < 1
$$
$$|x| < \lim_{n \rightarrow\infty}|x|\sqrt[n]{|a_n|} = R$$

## –û–±–æ–±—â–µ–Ω–Ω—ã–π —Å—Ç–µ–ø–µ–Ω–Ω–æ–π —Ä—è–¥
$$\sum_{n=0}^\infty a_n(x-x_o)^n$$

–î–ª—è –æ–±–æ–±—â–µ–Ω–Ω—ã—Ö —Å—Ç–µ–ø–µ–Ω–Ω—ã—Ö —Ä—è–¥–æ–≤ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –≤—Å–µ, —á—Ç–æ —Å–ø—Ä–∞–≤–µ–¥–ª–∏–≤–æ –¥–ª—è –æ–±—ã—á–Ω—ã—Ö —Å—Ç–µ–ø–µ–Ω–Ω—ã—Ö —Ä—è–¥–æ–≤.

$$|x-x_o| < R$$

### –ü—Ä–∏–º–µ—Ä: –ù–∞–π—Ç–∏ –æ–±–ª–∞—Å—Ç—å —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏ —Å—Ç–µ–ø–µ–Ω–Ω–æ–≥–æ —Ä—è–¥–∞
$$\sum_{n=1}^\infty \frac{x^n}n^2$$
$$u(x) = \frac{|x^n|}{n^2}$$
$$\lim_{n \rightarrow \infty} \sqrt[n]{\frac{|x^n|}{n^2}} = \lim_{n \rightarrow \infty} |x|\frac{1}{\sqrt[n]{n^2}} < 1$$

–ò–Ω—Ç–µ—Ä–≤–∞–ª –∞–±—Å–æ–ª—é—Ç–Ω–æ–π —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏ $(-1, 1)$.

–ò—Å—Å–ª–µ–¥—É–µ–º —Å—Ö–æ–¥–∏–º–æ—Å—Ç—å –Ω–∞ –∫—Ä–∞—è—Ö.
$$x=1, \ \sum_{n=1}^\infty \frac{1}{n^2}, \ p > 1, \ —Ä—è–¥ \ –î–∏—Ä–∏—Ö–ª–µ$$
$$x = -1, \ \sum_{n=1}^\infty \frac{(-1)^n}{n^2} \ —Ä—è–¥ \ —Å—Ö–æ–¥–∏—Ç—Å—è$$

–û–±–ª–∞—Å—Ç—å —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏: $[-1, 1]$.

## –°–≤–æ–π—Å—Ç–≤–∞ —Å—Ö–æ–¥—è—â–∏—Ö—Å—è —Å—Ç–µ–ø–µ–Ω–Ω—ã—Ö —Ä—è–¥–æ–≤.
–ò–Ω—Ç–µ—Ä–≤–∞–ª —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏ $x \in (-R, R)$.

1. –°—É–º–º–∞ —Å—Ç–µ–ø–µ–Ω–Ω–æ–≥–æ —Ä—è–¥–∞ $S(x) = \sum_{n=0}^\infty a_nx^n$ –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è –≤ –æ–±–ª–∞—Å—Ç–∏ —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏.

3. –°—Ç–µ–ø–µ–Ω–Ω–æ–π —Ä—è–¥ –º–æ–∂–Ω–æ –ø–æ—á–ª–µ–Ω–Ω–æ –∏–Ω—Ç–µ–≥—Ä–∏—Ä–æ–≤–∞—Ç—å.

## –†–∞–≤–Ω–æ–º–µ—Ä–Ω–∞—è —Å—Ö–æ–¥–∏–º–æ—Å—Ç—å —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ–≥–æ —Ä—è–¥–∞

$$\sum_{n=1}^\infty u_n(x)$$
$$S(x) = \sum_{n=1}^\infty u_n(x), \ x \in D, \ D - –æ–±–ª–∞—Å—Ç—å \ —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏$$
–ï—Å–ª–∏ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ $$|S(x) - s_n(x)|<\varepsilon, \forall x in E, E \leq D$$, —Ç–æ —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–π —Ä—è–¥ —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ —Å—Ö–æ–¥–∏—Ç—Å—è –≤ $E$.

–ü—É—Å—Ç—å $x \in (-R, R)$, $x_o \in (-R, R)$.
–ë—É–¥–µ–º —Å—á–∏—Ç–∞—Ç—å
$$|x-x_o| < \delta > 0$$

$$|S(x) - S(x_o)| = |S(x) - S_n(x)| + |S_n(x) - S_n(x_o)| - |S_n(x_o) - S(x_o)| \leq \dots$$
$$\dots \leq |S(x) - S_n(x)| + |S_n(x) - S_n(x_o)| + |S_n(x_o) - S(x_o)| \leq \varepsilon$$
–ö–∞–∂–¥–æ–µ —Å–ª–∞–≥–∞–µ–º–æ–µ –º–µ–Ω—å—à–µ $\frac\varepsilon3$.

### –ö—Ä–∏—Ç–µ—Ä–∏–π —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–π —Å—Ö–æ–¥–∏–º–æ—Å—Ç–∏ –ö–æ—à–∏

### –†—è–¥ –¢–µ–π–ª–æ—Ä–∞ (–ú–∞–∫–ª–æ—Ä–µ–Ω–∞)
–ü—É—Å—Ç—å $f(x)$ - —Ñ—É–Ω–∫—Ü–∏—è –∏–º–µ—é—â–∞—è –ø—Ä–æ–∏–∑–≤–æ–¥–Ω—ã–µ –ª—é–±–æ–≥–æ –ø–æ—Ä—è–¥–∫–∞ –≤ –æ–∫—Ä–µ—Å—Ç–Ω–æ—Å—Ç–∏ $(x_o - \delta, x_o + \delta)$ —Ç–æ—á–∫–∏ $x_0$. –¢–æ–≥–¥–∞ —Ä—è–¥ –¢–µ–π–ª–æ—Ä–∞ –≤ —ç—Ç–æ–π –æ–∫—Ä–µ—Å—Ç–Ω–æ—Å—Ç–∏.
$$\sum_{k=0}^\infty \frac{f^{(k)(x_o)}}{k!}(x-x_o)^k \ (5)$$

–ß–∞—Å—Ç–∏—á–Ω–∞—è —Å—É–º–º–∞ —Ä—è–¥–∞ –¢–µ–π–ª–æ—Ä–∞ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –º–Ω–æ–≥–æ—á–ª–µ–Ω–æ–º –¢–µ–π–ª–æ—Ä–∞ –∏ –∏–º–µ–µ—Ç –≤–∏–¥:

–û—Å—Ç–∞–ª—å–Ω–æ–µ, –ø–µ—Ä–µ—Ö–æ–¥–∏—Ç –≤ –æ—Å—Ç–∞—Ç–æ–∫ $r_n(x)$

–ï—Å–ª–∏ —Å–ø—Ä–∞–≤–µ–¥–ª–∏–≤–æ 6, —Ç–æ –≥–æ–≤–æ—Ä—è—Ç, —á—Ç–æ $f(x)$ —Ä–∞–∑–ª–∞–≥–∞–µ—Ç—Å—è –≤ —Ä—è–¥ –¢–µ–π–ª–æ—Ä–∞.
$$f(x) = \sum_{k=0}^\infty \frac{f^{(k)(x_o)}}{k!}(x-x_o)^k \ (6)$$

–ü—É—Å—Ç—å –Ω–µ–∫–∞—è —Ç–æ—á–∫–∞ $x \in (x_o - \delta, x_o + \delta)$, $f(x)$ –∏–º–µ–µ—Ç $n+1$ –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω—ã—Ö –ø—Ä–æ–∏–∑–≤–æ–¥–Ω—ã—Ö:

–ï—Å–ª–∏ $f(x)$ –∏–º–µ–µ—Ç –ø—Ä–æ–∏–∑–≤–æ–¥–Ω—ã–µ –ª—é–±–æ–≥–æ –ø–æ—Ä—è–¥–∫–∞ –≤ –æ–∫—Ä–µ—Å—Ç–Ω–æ—Å—Ç–∏ $x_o$ –∏ –æ—Å—Ç–∞—Ç–æ–∫ —Ñ–æ—Ä–º—É–ª—ã –¢–µ–π–ª–æ—Ä–∞ $r_n(0) \rightarrow 0|_{n \rightarrow \infty}$, —Ç–æ —Ñ—É–Ω–∫—Ü–∏—é –º–æ–∂–Ω–æ —Ä–∞–∑–ª–æ–∂–∏—Ç—å –≤ —Ä—è–¥ –¢–µ–π–ª–æ—Ä–∞.
