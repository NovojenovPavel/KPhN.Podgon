–í–æ–ª–Ω—ã —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω—è—é—â–∏–µ—Å—è –≤ —Å—Ä–µ–¥–µ –Ω–∞–∑—ã–≤–∞—é—Ç—Å—è __–ø—Ä–æ–¥–æ–ª—å–Ω—ã–º–∏__ –µ—Å–ª–∏ –∫–æ–ª–µ–±–∞–Ω–∏—è —á–∞—Å—Ç–∏—Ü –æ–∫–æ–ª–æ —Å–≤–æ–∏—Ö –ø–æ–ª–æ–∂–µ–Ω–∏–π —Ä–∞–≤–Ω–æ–≤–µ—Å–∏—è –ø—Ä–æ–∏—Å—Ö–æ–¥—è—Ç –≤–¥–æ–ª—å –Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–∏—è —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–∏—è –≤–æ–ª–Ω—ã –∏ __–ø–æ–ø–µ—Ä–µ—á–Ω—ã–º–∏__, –µ—Å–ª–∏ —ç—Ç–∏ –∫–æ–ª–µ–±–∞–Ω–∏—è –ø—Ä–æ–∏—Å—Ö–æ–¥—è—Ç –ø–µ—Ä–ø–µ–Ω–¥–∏–∫—É–ª—è—Ä–Ω–æ –Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–∏—é —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–∏—è –≤–æ–ª–Ω—ã.

–í–æ–ª–Ω–æ–≤–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –¥–ª—è –ø–ª–æ—Å–∫–æ–π –≤–æ–ª–Ω—ã —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω—è—é—â–µ–π—Å—è –≤–¥–æ–ª—å –æ—Å–∏ $x$:
$$\xi(x, t) = f(t-\frac{x}{v}), v - —Å–∫–æ—Ä–æ—Å—Ç—å \ —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–∏—è$$
$$t = 0, x = 0 \rightarrow \xi(0,0) = 0$$
$$t = t_1, x = vt_1 \rightarrow \xi(t_1, x_1) = f\left(t_1-\frac{vt_1}{v}\right) = \xi(0,0)$$

$\color{rgb(144, 238, 144)}–û–ø—Ä.$ –í–æ–ª–Ω—ã, –∫–æ—Ç–æ—Ä—ã–µ –æ–ø–∏—Å—ã–≤–∞—é—Ç—Å—è —Ñ—É–Ω–∫—Ü–∏–µ–π –∫–æ—Å–∏–Ω—É—Å–∞ –∏–ª–∏ —Å–∏–Ω—É—Å–∞, –Ω–∞–∑—ã–≤–∞—é—Ç—Å—è __–≥–∞—Ä–º–æ–Ω–∏—á–µ—Å–∫–∏–º–∏__.

–§—É–Ω–∫—Ü–∏—è –ª—é–±–æ–≥–æ –≤–∏–¥–∞ –º–æ–∂–µ—Ç –±—ã—Ç—å –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∞ –≤ –≤–∏–¥–µ —Å—É–ø–µ—Ä–ø–æ–∑–∏—Ü–∏–∏ –≥–∞—Ä–º–æ–Ω–∏—á–µ—Å–∫–∏—Ö —Ñ—É–Ω–∫—Ü–∏–π. –†–∞–∑–æ–±—Ä–∞–≤—à–∏—Å—å —Å –≥–∞—Ä–º–æ–Ω–∏—á–µ—Å–∫–∏–º–∏ –≤–æ–ª–Ω–∞–º–∏, —Ä–∞–∑–±–µ—Ä–µ–º—Å—è —Å–æ –≤—Å–µ–º–∏ –æ—Å—Ç–∞–ª—å–Ω—ã–º–∏.

$$\xi(x,t) = a \cos[\omega(t-\frac{x}{v})+\alpha]$$
$$\omega = 2\pi\nu,\:\: T = \frac{2\pi}\omega = \frac1\nu$$
$\color{rgb(144, 238, 144)}–û–ø—Ä.$ –î–ª–∏–Ω–Ω–∞ –≤–æ–ª–Ω—ã $\lambda$ - —Ä–∞—Å—Å—Ç–æ—è–Ω–∏–µ, –Ω–∞ –∫–æ—Ç–æ—Ä–æ–µ —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω—è–µ—Ç—Å—è –≤–æ–∑–º—É—â–µ–Ω–∏–µ –∑–∞ –æ–¥–∏–Ω –ø–µ—Ä–∏–æ–¥.
$$\lambda = vT$$
$$\xi(x,t) = a \cos[\omega t - kx +\alpha]$$
$$k = \frac\omega{\frac\lambda T} = \frac{2\pi}\lambda$$

–ï—Å–ª–∏ —Å—Ä–µ–¥–∞ –∑–∞–±–∏—Ä–∞—é—Ç —ç–Ω–µ—Ä–≥–∏—é —É –≤–æ–ª–Ω—ã, –∞–º–ø–ª–∏—Ç—É–¥–∞ —É–±—ã–≤–∞–µ—Ç –ø–æ —ç–∫—Å–ø–æ–Ω–µ–Ω—Ü–∏–∞–ª—å–Ω–æ–º—É –∑–∞–∫–æ–Ω—É:
$$\xi(x,t) = a_o e^{-\gamma t} \cos[\omega t - kx +\alpha]$$

$\color{rgb(144, 238, 144)}–û–ø—Ä.$ __–í–æ–ª–Ω–æ–≤–∞—è –ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç—å__ - –≥–µ–æ–º–µ—Ç—Ä–∏—á–µ—Å–∫–æ–µ –º–µ—Å—Ç–æ —Ç–æ—á–µ–∫, –∫–æ–ª–µ–±–ª—é—â–∏—Ö—Å—è –≤ –æ–¥–∏–Ω–∞–∫–æ–≤–æ–π —Ñ–∞–∑–µ.

$\color{rgb(144, 238, 144)}–û–ø—Ä.$ __–í–æ–ª–Ω–æ–≤–æ–π —Ñ—Ä–æ–Ω—Ç__ - –≥–µ–æ–º–µ—Ç—Ä–∏—á–µ—Å–∫–æ–µ –º–µ—Å—Ç–æ —Ç–æ—á–µ–∫, –¥–æ –∫–æ—Ç–æ—Ä–æ–≥–æ –¥–æ—Ö–æ–¥—è—Ç –∫–æ–ª–µ–±–∞–Ω–∏—è –≤ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–π –º–æ–º–µ–Ω—Ç –≤—Ä–µ–º–µ–Ω–∏.
